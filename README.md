# Judgement Analysis System

AI-powered legal judgement analysis system using NLP and RAG.

## Features
- User authentication
- Case document extraction
- Legal domain classification 
- AI-powered judgment summarization
- Fast Legal Search using FAISS
- User notes
- Dashboard(Data Storage)
- Semantic case retrieval
- Bookmarking important cases

## Tech Stack
- Programming Language
- Python
- Web Technologies
- HTML
- CSS
- JavaScript
- Backend Framework
- Flask
- Database
- MongoDB
- ML Libraries
- PyTorch
- Scikit-learn
- NLP
- Transformers
- Sentence Embeddings
- Retrieval-Augmented Generation (RAG)
- Large Language Model (LLM)
- Grok API (LLM for generating legal insights and summaries)
- Vector Search
- FAISS (Facebook AI Similarity Search)

## Datasets
- Raw dataset: 26,690 legal cases
- Processed dataset: 2000 cases
- Vector index: 2000 embeddings

## Run Project
- pip install -r requirements.txt
- python app.py
